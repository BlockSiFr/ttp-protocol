# Trust Proof Model
Defines threshold, attestation, delegation, route, and composite trust proof envelopes.

## proofMode
`plain`, `signed`, `zk-placeholder`, `zk`.

## Canonicalization
Stable key order, ISO-8601 timestamps, normalized subject/action/resource strings, fixed numeric precision.

## Privacy goal
Prove sufficient trust without exposing full trust history.

## Non-goal
Platform adapter mapping.
