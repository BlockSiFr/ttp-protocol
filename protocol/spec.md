# Protocol Spec Note

The active Trust Transfer Protocol draft is maintained at [`../SPECIFICATION.md`](../SPECIFICATION.md).

This `protocol/` directory contains earlier token, receipt, aggregation, schema, and test-vector work that may inform future signed-claim and runtime integration phases. It should be treated as reference material, not the current stable protocol contract.

Current status:

- Protocol specification draft complete.
- Reference implementation in active development.
- Current milestone: MVP parser + trust decay evaluator.
- Production use: not yet recommended.
